# 图片目录

把照片放在这里，然后在 `../data.js` 对应的条目里引用：

```js
cover: "images/your-photo.jpg",                  // 单张封面
images: ["images/a.jpg", "images/b.jpg"],        // 多张（详情页显示可点图廊）
```

建议：封面用竖图（约 3:4 / 4:5）在瀑布流里最好看；文件用小写英文名，避免空格。

## ⚠️ 只放你有权使用的图片
- ✅ 你**自己拍的** / 你**自己在小红书发布**的照片（你拥有版权）
- ✅ 免费可商用图库 **Unsplash / Pexels**
- ✅ 商家**官方授权**的图
- ❌ 直接搬运**别人**小红书 / 其它网站的图 —— 侵权，且小红书有防盗链，外链会裂图
